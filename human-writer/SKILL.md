# Human Writer

Strip AI patterns from writing. Make it sound like a person wrote it.

## The Problem

AI writing has tells. Readers (and detectors) spot them. This skill eliminates those patterns.

## Quick Check: Does It Sound AI?

Scan for these red flags:

### Words That Scream AI
- "Delve" / "delve into"
- "Furthermore," "Moreover," "Additionally," "In addition"
- "It's important to note," "It's worth mentioning"
- "In today's [world/landscape/era]"
- "Certainly," "Absolutely," "Great question"
- "Utilize" (just say "use")
- "Leverage" (as a verb)
- "Facilitate," "Commence," "Endeavor"
- "Robust," "Seamless," "Cutting-edge," "Game-changing"
- "Landscape" (for anything other than actual land)
- "Dive into" / "Let's dive in"
- "Unpack" (for explaining)
- "At the end of the day"

### Phrases That Give It Away
- "From X to Y" ("From beginners to experts," "From bustling cities to serene landscapes")
- "X isn't just Y; it's Z"
- "Whether you're X or Y, this Z..."
- "In this article, we will..."
- "As we discussed earlier..."
- Vague attribution: "Some have said," "Many believe," "It has been described as"
- Importance-inflation: "a titan of," "a beacon of," "a cornerstone of"

### Structural Tells
- **Bold header:** followed by colon, then explanation (like this line)
- Bullet lists for everything
- Every paragraph: topic sentence → evidence → neat summary
- Em dashes everywhere — like this — breaking up thoughts
- Uniform paragraph lengths (all roughly the same size)
- Uniform sentence lengths (no variation in rhythm)
- Restating the question before answering
- Summary paragraph at the end that adds nothing

### Tone Tells
- Perpetually balanced ("on the one hand... on the other hand") when a clear stance is appropriate
- Excessive hedging: "may," "might," "could potentially," "it's possible that"
- No personal voice or opinion
- Promotional without substance ("revolutionary," "transformative," "game-changing")
- Flowery descriptions instead of specific facts

## The Fix: Make It Human

### 1. Vary Your Rhythm
Humans don't write in uniform sentences. Mix it up.

Bad (AI):
```
The platform provides comprehensive analytics. Users can track their performance metrics. The dashboard displays real-time data. Teams can collaborate on insights.
```

Better (human):
```
The analytics are solid. You get real-time metrics, and the dashboard actually makes sense—no PhD required. Teams can poke around together.
```

### 2. Take a Stance
AI hedges. Humans have opinions.

Bad (AI):
```
There are various approaches to this problem. Some prefer method A, while others find method B more suitable. The best choice depends on your specific needs.
```

Better (human):
```
Method A works better for most cases. Method B exists, but unless you're doing X, skip it.
```

### 3. Be Specific, Not Flowery
AI inflates importance. Humans cite facts.

Bad (AI):
```
Ray Kroc was a visionary titan of the casual culinary landscape who transformed the restaurant industry.
```

Better (human):
```
Ray Kroc turned McDonald's from a single restaurant into 40,000 locations.
```

### 4. Use Normal Words
Anglo-Saxon beats Latinate. Short beats long.

| AI Word | Human Word |
|---------|------------|
| Utilize | Use |
| Facilitate | Help |
| Commence | Start |
| Endeavor | Try |
| Possess | Have |
| Regarding | About |
| Demonstrate | Show |
| Numerous | Many |
| Purchase | Buy |
| Obtain | Get |

### 5. Cut the Scaffolding
AI loves meta-commentary. Humans just say the thing.

Bad (AI):
```
In this section, we'll explore the key factors to consider. First, let's examine the most important aspect. As we'll see, this has significant implications.
```

Better (human):
```
Here's what matters: [just say it]
```

### 6. Use Contractions
"It is" → "It's"
"Do not" → "Don't"
"We will" → "We'll"

AI often avoids contractions. Humans use them constantly.

### 7. Let Sentences Breathe
Not every sentence needs a transition. Sometimes you just start a new thought.

Bad (AI):
```
The system is fast. Furthermore, it is reliable. Additionally, it scales well. Moreover, the pricing is competitive.
```

Better (human):
```
The system is fast and reliable. Scales well too. Pricing won't kill you.
```

### 8. Kill the Summary Paragraph
If you said it, you said it. Don't recap.

Bad (AI):
```
[Three paragraphs of content]

In conclusion, we've explored how X relates to Y, examined the key factors of Z, and discussed the implications for your workflow. By following these guidelines, you'll be well-positioned to achieve success.
```

Better (human):
```
[Three paragraphs of content]

[Just end. The reader got it.]
```

## Checklist Before Publishing

- [ ] No "delve," "utilize," "leverage," "landscape," "robust," "seamless"
- [ ] No "Furthermore," "Moreover," "Additionally" chains
- [ ] No "It's important to note" or "It's worth mentioning"
- [ ] Sentence lengths vary (short, medium, long, mixed)
- [ ] Paragraph lengths vary
- [ ] Has at least one clear opinion or stance
- [ ] Specific facts/numbers instead of flowery descriptions
- [ ] Uses contractions naturally
- [ ] No unnecessary summary at the end
- [ ] Doesn't restate the question/prompt
- [ ] Would pass the "did a person write this?" sniff test

## When Reviewing Text

1. Read it aloud. Does it sound like speech or a press release?
2. Count the AI words. More than 2? Rewrite.
3. Check paragraph lengths. All the same? Vary them.
4. Find the opinion. None? Add one or acknowledge you're being neutral deliberately.
5. Cut 20%. AI overwrites. Humans edit.
