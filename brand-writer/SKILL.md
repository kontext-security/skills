# Brand Writer for Kontext.dev

Write content that reflects Kontext's voice: optimistic, technically confident, clear, and empowering. Never fear-based.

## Quick Reference

**Core message**: "From demo to deployment. For agents."
**What we do**: Close the gap between agent demos that work and agent deployments that don't
**Voice**: Innovative Disruptor—optimistic, developer-first, technically confident
**Avoid**: FUD, buzzwords ("trust layer" without context), enterprise-speak, abstract claims

## Workflow

### 1. Identify Content Type
- **Landing page / Hero**: See [content-patterns.md](references/content-patterns.md) → Headlines, Hero Sections
- **Product description**: See [content-patterns.md](references/content-patterns.md) → Product/Feature Descriptions
- **Blog / Thought leadership**: See [content-patterns.md](references/content-patterns.md) → Blog Posts
- **Email**: See [content-patterns.md](references/content-patterns.md) → Marketing Emails
- **CTA / Tagline**: See [content-patterns.md](references/content-patterns.md) → CTAs, Taglines

### 2. Load Context
Read [brand-foundation.md](references/brand-foundation.md) for:
- Voice principles and culture values
- Core messaging framework (problem → shift → solution → promise)
- Vocabulary preferences

For inspiration and anti-patterns, see [inspiration.md](references/inspiration.md).

### 3. Write or Refine

**When generating new content**:
1. Start with the outcome the reader cares about
2. Ground abstract claims with concrete specifics
3. Use active voice and direct address ("you")
4. Include speed/ease signals when relevant ("in minutes")

**When refining existing content**:
1. Check against voice principles (optimistic? confident? clear?)
2. Remove FUD, buzzwords, passive voice
3. Sharpen the hook—first line must earn attention
4. Ensure CTA is specific and action-oriented

### 4. Self-Check

Before delivering, verify:
- [ ] Opens with outcome or transformation, not features
- [ ] No fear-based framing (threats, attacks, risks as primary hook)
- [ ] No empty buzzwords (cutting-edge, next-gen, revolutionary)
- [ ] Concrete over abstract (examples, metrics, specifics)
- [ ] Active voice, direct address
- [ ] CTA is clear and action-oriented
